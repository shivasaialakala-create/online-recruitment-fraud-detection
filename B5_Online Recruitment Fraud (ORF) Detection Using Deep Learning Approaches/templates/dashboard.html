Project Overview

This system uses **Deep Learning** and traditional Machine Learning approaches (Random Forest, Decision Tree, CNN) to detect **Online Recruitment Fraud (ORF)** in job postings. The goal is to safeguard job seekers from malicious schemes by providing real-time, high-accuracy threat detection.

Current Fraud Detection Status

Total Postings Analyzed: 15,480

Confirmed Fraudulent: 721 (4.66%)

Current Model (CNN) Accuracy: 98.2%

System is running in real-time, providing proactive security. Click 'Prediction Tool' to analyze a new posting.

Upcoming Features

We are constantly improving the system. Next updates will include:

  • LLM Integration: Using large language models for contextual risk scoring.
  • User Reporting: A feature for users to report new suspicious job postings.
  • Visualization Dashboard: Enhanced charts for model performance over time.

Model Benchmark (F1-Scores)

Random Forest: 0.85

Decision Tree: 0.81

Convolutional Neural Network (CNN): 0.94

The CNN model demonstrates superior performance in classifying job postings due to its ability to capture complex patterns in text data.
